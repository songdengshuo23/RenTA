# 路由分类器测试报告

## 一、测试说明

### 1.1 路由分类器

```
输入: 自然语言任务描述
输出: {"LLM": float, "Agent": float, "多Agent": float}
模型: DeepSeek Chat (deepseek-chat)
```

### 1.2 三档定义

| 类别 | 定义 | 核心判断 |
|------|------|----------|
| **LLM** | 直接大模型回答，不需要任何外部工具 | 纯知识/推理/改写/创意写作 |
| **Agent** | 需要工具，但**一个职业方向的人**能独立完成 | 需要搜索/代码/文件/API |
| **多Agent** | 需要**至少两个不同职业方向的人**分工协作 | 跨领域/架构级/独立验证 |
